อพาร์ตเมนต์

อพาร์ตเมนต์ให้เช่ารายเดือนเปิดใหม่ ขนาดเล็ก เพียง 46 ห้อง ครบครันทั้งเฟอร์นิเจอร์บิวอินน์ แอร์ น้ำอุ่น ตู้เย็น พร้อมที่จอดรถ ตั้งอยู่กลางใจเมือง
ห่างจากถนนใหญ่ (ถ.ประชาสโมสร) เพียง 50 ม.

สิ่งอำนวยความสะดวก

ทุกห้องตกแต่งเฟอร์นิเจอร์ครบพร้อมเข้าอยู่ แอร์ ทีวี พัดลม ตู้เย็น ไมโครเวฟ เครื่องทำน้ำอุ่น ส่วนกลางมีระบบรักษาความปลอดภัย 24 ชม คีย์การ์ด
กล้องวงจรปิด ที่จอดรถ

รายละเอียดการเช่า

รายคาเช่า : 3500 - 5000 บาท/เดือน
เงินมัดจำ : 2000 บาท
จ่ายล่วงหน้า : 1 เดือน
อินเตอร์เน็ต : ฟรี

สถานที่ใกล้เคียง

ตั้งอยู่กลางใจเมือง ห่างจากถนนใหญ่ (ถ.ประชาสโมสร) เพียง 50 ม. สามารถทะลุออกถนนหลังเมืองได้ อยู่กึ่งกลางสวนสาธารณะ 2 แห่ง คือ บึงทุ่งสร้าง และ บึงแก่นนคร ใกล้ศูนย์ราชการ ตลาดนัดถนนคนเดิน โรงพยาบาลศูนย์ขอนแก่น ตลาดโต้รุ่ง มีรถประจำทางผ่านหลายสาย หน้าปากทางเป็นจุดจอดรถรับส่งบขส
